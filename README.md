# 弈衡五子棋 - 比赛运行版

支持2026省赛五子棋项目规则的五子棋智能博弈程序。

## 主要功能

- **指定开局**：支持26种官方指定开局
- **三手交换**：程序自动判断是否交换
- **五手N打**：N=2/3/4/5可选
- **禁手检测**：黑棋三三、四四、长连禁手可视化
- **双人对弈**：双方各15分钟计时

## 运行方式

```bash
# 进入目录
cd "弈衡五子棋_比赛运行版"

# 运行图形界面（推荐）
python competition_gui.py

# 或双击运行
run_gui.bat
```

## 规则说明

- 黑1必须为H8（天元）
- 白2必须在H8的8个紧邻点
- 黑3必须在中心5×5区域（F6-J10）
- 黑棋存在三三、四四、长连禁手
- 白棋无禁手
